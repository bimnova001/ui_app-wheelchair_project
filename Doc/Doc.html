Overview

ระบบควบคุมหุ่นยนต์นี้สร้างด้วย Python + CustomTkinter รองรับการขยายฟังก์ชันและสถานะได้ง่าย
Sidebar สำหรับปุ่มฟังก์ชันและตัวเลือก, Status Panel แสดงสถานะการทำงานแบบ real-time
รองรับการเพิ่มฟังก์ชัน/checkbox ใหม่ได้ทันที เพียงเพิ่มใน list ในโค้ด

Function Status ( ตัวอย่างการใช้งาน )

Connect
ตรวจสอบ Python ในระบบ
CMD: python --version
Settings
ทดสอบคำสั่ง echo (หรือ cmd echo บน Windows)
CMD: echo settings ok
Shutdown
Ping localhost 1 ครั้ง
CMD: ping 127.0.0.1 -n 1
สถานะไฟเขียว = ทำงานได้ปกติ
สถานะไฟแดง = มี error หรือไม่ตอบสนอง

Checkbox Status ( ตัวอย่างการใช้งาน )

Enable notifications
เปิด/ปิดการแจ้งเตือน
Auto-connect
เชื่อมต่ออัตโนมัติเมื่อเริ่มต้น
Show battery
แสดงสถานะแบตเตอรี่
สถานะ checkbox จะเปลี่ยนสีตามค่าปัจจุบัน (ติ๊กถูก = เขียว, ไม่ติ๊ก = แดง)

Usage Example ( ตัวอย่างการใช้งานใน )

เพิ่ม/แก้ไขฟังก์ชัน

self.func_list = [
    {"name": "Connect", "cmd": ["python", "--version"]},
    {"name": "Settings", "cmd": ["echo", "settings ok"]},
    {"name": "Shutdown", "cmd": ["ping", "127.0.0.1", "-n", "1"]},
    # เพิ่มฟังก์ชันใหม่ได้ที่นี่
]

เพิ่ม/แก้ไข Checkbox

self.checkbox_list = [
    {"name": "Enable notifications"},
    {"name": "Auto-connect"},
    {"name": "Show battery"},
    # เพิ่ม checkbox ใหม่ได้ที่นี่
]

เรียกใช้ฟังก์ชันจากชื่อ

def call_func_by_name(self, func_name):
    func = next((f for f in self.func_list if f["name"] == func_name), None)
    if func:
        self.on_func_click(func)
        # หรือจะใช้ self.set_func_status(func_name, True/False) ก็ได้

ตัวอย่างการใช้งานใน Checkbox

def on_checkbox_toggle(self, name):
    value = self.checkbox_vars[name].get()
    self.set_checkbox_status(name, value)
    if name == "Enable notifications" and value == 0:
        self.call_func_by_name("Connect")

How to Extend ( ตัวอย่างการใช้งานใน )

  • เพิ่มฟังก์ชันใหม่ใน self.func_list ได้ทันที (กำหนดชื่อและคำสั่ง subprocess)
  • เพิ่ม checkbox ใหม่ใน self.checkbox_list ได้ทันที
  • สถานะของแต่ละฟังก์ชันและ checkbox จะอัปเดตอัตโนมัติ
  • สามารถนำ logic ใน on_func_click และ on_checkbox_toggle ไปต่อยอดกับ hardware หรือ API จริงได้
